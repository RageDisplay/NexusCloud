# NexusCloud - NAS Сервис

### Шифрование данных
- **AES-256-GCM**: Шифрование данных в покое (at rest)
- **TLS 1.3**: Шифрование при передаче (in transit)
- **SHA-256**: Хеширование файлов для проверки целостности

### Аутентификация
- **bcrypt**: Хеширование паролей с cost=14
- **JWT токены**: Безопасные сеансы с автоистечением
- **Проверка попыток входа**: Защита от brute-force атак
- **Rate limiting**: Ограничение количества запросов

### Аудит и логирование
- **Полное логирование**: Все операции записываются в БД
- **IP адреса**: Отслеживание источника действий
- **User Agent**: Информация о клиентах
- **Статусы операций**: Успешные и ошибочные операции

## Требования

- Docker и Docker Compose
- Go 1.24+ 
- OpenSSL 
- SSD диск для хранения
